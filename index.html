Pen: CSS 3D SOLAR SYSTEM

By Julian Garnier

This wonderful pen simulates our solar system with CSS. As you can see this pen shows the planets revolving around the sun. You can select each planet using the navigation bar at the bottom, and it will show you the Orbit Velocity in km/h.

See the Pen CSS 3D Solar System by Julian Garnier (@juliangarnier) on CodePen.
